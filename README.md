# Travel & Tourism Management System

## 📌 Project Overview
The Travel & Tourism Management System is a console-based application developed using Core Java. It helps manage travel bookings, customer details, hotel reservations, vehicle rentals, and billing operations efficiently.

## 🚀 Features
- User Authentication
- Customer Management
- Travel Package Booking
- Hotel Reservation
- Vehicle Rental
- Billing System
- Simple Menu-Driven Interface

## 🛠️ Technologies Used
- Java
- Object-Oriented Programming (OOP)
- VS Code
- Git & GitHub

## 📂 Project Structure
Travel-Tourism-Management-System/
├── AuthManager.java
├── BillingSystem.java
├── HotelBooking.java
├── PlaceModule.java
├── TouristUser.java
├── TourismApp.java
└── VehicleRentalApp.java

## ▶️ How to Run
1. Clone this repository.
2. Open the project in VS Code or Eclipse.
3. Compile the Java files.
4. Run `TourismApp.java`. or 'java TourismApp'.
